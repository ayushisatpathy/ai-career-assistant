# AI Career Assistant

AI Career Assistant is a full-stack web application that uses Generative AI to help users improve their resumes and prepare for technical interviews.

The objective of this project is not only to build an AI-powered application, but also to document the engineering process behind building software with AI assistance. Alongside the implementation, this repository captures architectural decisions, project planning, AI collaboration, and reflections throughout development.

## Development Philosophy

This project is being developed incrementally.

Each feature follows the same workflow:

1. Understand the requirement
2. Explore alternative approaches
3. Implement the solution
4. Test the implementation
5. Document architectural decisions
6. Reflect on AI usage and lessons learned

The objective is to treat AI as an engineering assistant rather than a replacement for engineering judgment.
